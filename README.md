## 🧠 My Approach

For every lab I follow the same workflow:

1. **Understand the vulnerability** — what it is, why it exists, how it works
2. **Analyze the target** — find the injection point, test behavior
3. **Build the payload** — craft the attack step by step
4. **Document everything** — write notes I can reference later
5. **Extract the lesson** — what would a defender do to prevent this?

> ⚠️ All content is for **educational purposes only**. No full solutions are shared — only methodology, payloads, and analysis.
